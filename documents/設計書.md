## 4. デザイン仕様

### 4-1. グローバル

- **背景色** : `#ffffff`（全セクション統一）
- **角丸** : 全図形に `border-radius: 20px` 以上を適用
- **フォント** :
  - 見出し → `Playfair Display`（Google Fonts・セリフ体）
  - 本文　 → `Inter`（Google Fonts・サンセリフ体）
- **カラーパレット** :

| 変数名 | 値 | 用途 |
|--------|-----|------|
| `--text` | `#1a1a1a` | メインテキスト |
| `--text-muted` | `#6b7280` | サブテキスト・ラベル |
| `--accent` | `#6B4226` | アクセント（カカオブラウン） |
| `--surface` | `#faf8f5` | セクション背景（薄ベージュ） |
| `--surface-dark` | `#2d2419` | ダークカード背景 |
| `--border` | `#e8e2da` | 境界線 |

---

### 4-2. ナビゲーション

- 画面上部に固定（`position: sticky`）
- ロゴ「Cacao」を左端、セクションリンクを右端に配置
- `backdrop-filter: blur` によるすりガラス効果

## 5. セクション別仕様

### 5-1. Top（ヒーロー）

#### レイアウト

```
┌───────────────────────────────────────┐
│  [左カラム 45%]    [右カラム 55%]    │
│                                       │
│  I'm Cacao         [dish_cacao.png]   │
│  （小さなテキスト）                  │
│                                       │
└───────────────────────────────────────┘
```

#### アニメーション仕様

| 要素 | 初期状態 | 最終状態 | 方向 |
|------|----------|----------|------|
| テキスト | `translateY(-60px)` `opacity: 0` | `translateY(0)` `opacity: 1` | **上 → 下** |
| 画像 | `translateY(60px)` `opacity: 0` | `translateY(0)` `opacity: 1` | **下 → 上** |

- `animation-duration` : `0.9s`
- `animation-timing-function` : `cubic-bezier(0.25, 0.46, 0.45, 0.94)`
- テキスト delay : `0.3s` / 画像 delay : `0.5s`

### 5-4. Portfolio（スタックカードホバー）

- **グリッド** : 2 × 2（計 4 アイテム）
- **各アイテム** : 3 枚のカードをスタック
- **テキスト** : 各最前面カードの中央に `"Coming Soon"`

#### カードアニメーション仕様

```
【通常時（スタック状態）】
  .card:nth-child(3) → rotate(-8deg) translateY(6px)   ← 最背面
  .card:nth-child(2) → rotate(-4deg) translateY(3px)
  .card:nth-child(1) → rotate(0deg)                    ← 最前面（Coming Soon）

【ホバー時（扇状展開）】
  .card:nth-child(3) → rotate(-20deg) translate(-44px, 12px)
  .card:nth-child(2) → rotate(-8deg)  translate(-16px, 6px)
  .card:nth-child(1) → rotate(8deg)   translate(20px, -6px)
```

- `transition` : `0.4s cubic-bezier(0.25, 0.46, 0.45, 0.94)`

## 7. JavaScript 仕様

### 7-1. Introduction 言語切替

```javascript
// 英日切替ロジック（フェードトランジション）
translateBtn.addEventListener('click', () => {
  introText.classList.add('fade-out')          // 0.3s フェードアウト
  setTimeout(() => {
    isJapanese = !isJapanese
    introText.textContent = isJapanese ? jaText : enText
    translateBtn.textContent = isJapanese
      ? 'Translate into English'
      : 'Translate into Japanese'
    introText.classList.remove('fade-out')     // フェードイン
  }, 300)
})
```

### 7-2. スクロール連動アニメーション

- `IntersectionObserver` でセクションが画面内に入ったタイミングで `.visible` クラスを付与
- `.visible` 付与時: `opacity: 0` + `translateY(20px)` → `opacity: 1` + `translateY(0)`

### 7-3. ナビゲーション アクティブ状態

- スクロール位置に応じて現在表示中セクションの nav リンクに `.active` クラスを付与
